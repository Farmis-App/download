## FARMIS v1.0.5

This release focuses on the crop search experience, a refreshed app icon, and a smaller install size thanks to Android ProGuard + resource shrinking. v1.0.5 installs over v1.0.4 in place — no uninstall required.

### What's new

- **Faster crop search.** Tap the header search to filter inline, with a floating filter bar that stays out of your way while scrolling.
- **Cleaner filters.** Category and season filters now use compact dropdown pills that open a bottom-sheet picker, replacing the old chip row.
- **About screen contact section.** Added contact details so you can reach the FARMIS team directly from inside the app.
- **New app icon.** Updated icon and Android adaptive icon for a sharper look on the home screen.
- **Smaller download.** Enabled ProGuard and Android resource shrinking, reducing APK size.

### Polish

- Profile tab uses tighter spacing and right-sized icons to match the rest of the app.
- Field screens now use the global theme tokens, fixing minor color inconsistencies in light and dark mode.

### How to upgrade

1. Download `farmis.apk` from this page.
2. Tap the file to install — it will upgrade your existing FARMIS app in place.
3. Open FARMIS and continue as before.

Full install instructions, including Tok Pisin walkthrough: https://farmis-app.github.io/download/
