## FARMIS v1.1.1

v1.1.1 is a packaging and distribution update that prepares FARMIS for its Google Play launch. The download is now much smaller, and the app is signed with FARMIS's own release key.

> ⚠️ **Please uninstall your existing FARMIS app before installing v1.1.1.** This release moves FARMIS to a new app identity, so Android treats it as a fresh install rather than an in-place upgrade. If you don't remove the old version first, you may end up with two FARMIS apps on your device.

### What's new

- **Smaller download.** The APK is now ~60 MB (down from ~104 MB) — faster to download and lighter on storage. We dropped emulator-only builds and ship optimized ARM packages for real phones.
- **Properly signed release.** v1.1.1 is signed with FARMIS's production release key, enabling secure updates going forward and Google Play distribution.
- **Google Play ready.** This is the first build prepared for the Google Play Store.

### How to install

1. Uninstall your current FARMIS app (long-press the icon → Uninstall).
2. Download `farmis.apk` from this page.
3. Tap the file to install.
4. Open FARMIS and continue as before.

Full install instructions, including Tok Pisin walkthrough: https://farmis-app.github.io/download/
